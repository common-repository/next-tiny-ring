Next Tiny Ring allows you to propose an advertising solution through display of banner ads shared on a network of several websites.
Choose the banner ads to be displayed and the websites where to add them.
Configure the style of the banners and the ads to be displayed.
Manage your banner ads (create new banners, modify or delete).
Usefull to increase your SEO between your websites creating entering links.

== Description ==
Next Tiny Ring allows you to propose an advertising solution through display of banner ads shared on a network of several websites.
Choose the banner ads to be displayed and the websites where to add them.
Configure the style of the banners and the ads to be displayed.
Manage your banner ads (create new banners, modify or delete).
Usefull to increase your SEO between your websites creating entering links.

== Languages ==
This plugin is available in English & French.

== Installation ==
1. Upload the plugin files to the '/wp-content/plugins/next-tiny-ring' directory, or install the plugin through the WordPress plugin screen directly.
3. Activate the plugin through the 'Plugins' screen in WordPress
2. Go to Next Tiny Ring menu to configure the plugin and save your changes.

== Frequently Asked Questions ==
= Why won't this work? =
It does work. If not just let me know! ;-)

= How can I create the database? =
You can use phpMyAdmin for example to create an EMPTY external database somewhere on your server.
Remember the name, user and password of this database as you will need to enter them in the plugin to connect it.

= Can I choose the websites to display its ads or not? =
Yes, you can!

= Can I choose the pages where to display ads? =
Yes, you can enter some page Ids to display ads.

= Can I exclude somes pages to not display ads? =
Yes, you can enter some page Ids to exclude them from displaying ads.
